## Features

- **Day & week views** for a quick, organized overview
- **Hourly task slots** with **descriptions** for important details
- **Edit & delete** to keep schedules flexible and up to date
- **Color‑coding** for categories like work, personal, and academic
- **Dashboard** to see a single day or the entire week at a glance
- **Local save** and **print** options for accessibility and offline use
- **Smooth navigation** between days and fast task creation
